# IoT-Based-Coal-Mine-Safety-Monitoring-and-Real-Time-Hazard-Alert-System-
This project introduces an IoT-Based Coal Mine Safety Monitoring and Intelligent Hazard 
Alert System aimed at improving real-time safety in subterranean mining settings. The system 
incorporates an ESP32 microcontroller alongside various sensors, including MQ-4 (methane), 
MQ-7 (carbon monoxide), MQ-2 (smoke and LPG gas), MQ-135, an additional air quality 
sensor, an ozone sensor, a PM2.5 dust sensor, a DHT11 (temperature and humidity), and an 
infrared temperature sensor for ongoing environmental monitoring. 
Sensor data is conveyed by Wi-Fi to a web-based monitoring dashboard (CoalGuard platform), 
which presents real-time readings, system status, and graphical trends. Upon connection via a 
data cable, real-time graphs are produced, and data is recorded in CSV format for study. The 
system comprises auditory alerts, buzzer alarms, and LED indications for prompt hazard 
detection. The ESP32-CAM module facilitates real-time video surveillance, permitting remote 
visual assessment of mining conditions. 
A threshold-based detection system identifies hazards like gas leaks, smoke, substandard air 
quality, and anomalous temperature. The system operates on a high-capacity lithium-ion 
battery, guaranteeing dependable and portable functionality. It can be further augmented by the 
application of Machine Learning models to anticipate hazards proactively. 
The system is economical, scalable, and markedly enhances safety, monitoring, and decision
making in intelligent mining contexts.
